IMPORTANTE: Para compilar entradas muito grandes, como é o caso da entrada de 1.000.000,
é necessário aumentar a quantidade de memória RAM fornecida do computador para o programa feito em C.


SEM MAKEFILE:

1. Vá para a pasta KNN_C:
cd .\KNN_C\

2. Compile com o GCC:
gcc -o ../compiles/main main.c leitura.c utilidade.c knn.c
../compiles/main


COM MAKEFILE:

1. Vá para a pasta KNN_C:
cd .\KNN_C\

2.Utilize o comando "all_multi" (multi para utilizar o multi threading) do makefile (ele compila e roda):
make all_multi
